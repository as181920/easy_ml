## Installation

### Bundler

Add this line to your application's Gemfile:

```ruby
  gem "easy_ml", git: "git@github.com:as181920/easy_ml.git", branch: "main"
```

### Gem install

```bash
  # gem install easy_ml
  bundle exec rake install
```
